# 🎬 Netflix Clone

A responsive Netflix landing page clone built using **HTML** and **CSS**. This project recreates the look and feel of Netflix's homepage, focusing on modern UI design, responsive layouts, and clean code structure.

## 🚀 Features

* Responsive design for desktop, tablet, and mobile devices
* Modern Netflix-inspired user interface
* Hero section with call-to-action
* Multiple content sections
* Clean and organized code structure
* Pure HTML & CSS (No JavaScript)

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Project Structure

```text
Netflix-Clone/
│
├── index.html
├── style.css
├── assets/
└── README.md
```

## ⚡ Getting Started

### Clone the repository

```bash
git clone https://github.com/karanpandit8914/Netflix-Clone
```

### Open the project folder

```bash
cd Netflix-Clone
```

### Run the project

Open `index.html` in your preferred web browser.
